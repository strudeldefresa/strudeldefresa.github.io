Sistema Nervioso Central (SNC)

  • Ansiolíticos: Reducen ansiedad (Diazepam).
  • Hipnóticos: Inducen sueño (Zolpidem).
  • Analgésicos: Reducen dolor (Morfina).

Sistema Nervioso Autónomo (SNA)

  • Simpaticomiméticos: Adrenalina.
  • Parasimpaticolíticos: Atropina.

Sistema Cardiovascular

  • Antihipertensivos: Enalapril.
  • Antiarrítmicos: Amiodarona.

Sistema Respiratorio

  • Broncodilatadores: Salbutamol.
  • Antitusivos: Codeína.

Sistema Digestivo

  • Antiácidos: Omeprazol.
  • Antieméticos: Ondansetrón.

Sistema Endocrino

  • Hipoglucemiantes: Metformina.
  • Corticoides: Prednisona.

Sistema Inmunológico

  • Antibióticos: Amoxicilina.
  • Antivirales: Aciclovir.

Antineoplásicos (anticancerígenos)

  • Citostáticos: Cisplatino.
  • Inmunoterapia: Nivolumab.

Farmacología Especial

  • Pediátrica, Geriátrica, Embarazo y Lactancia.
